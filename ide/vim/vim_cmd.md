# Vim command

## Mode command
|command|description|
|-------|-----------|
|:set list|show invisible characters|
|:set nolist|hide invisible characters|
|`/<pattern>\c`|case insensitive|
|`/<pattern>\C`|case sensitive|
|:echo @% |directory/name of the current file from the working directory|
